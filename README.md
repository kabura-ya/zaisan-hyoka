# zaisan-hyoka

課税明細書CSVを読み込み、国税庁「財産評価基準書」サイトから路線価図・倍率表PDFを一括ダウンロードするPythonスクリプトです。

## できること

- 課税明細書CSVを読み込み、住所から都道府県・市区町村・大字を自動判定
- **路線価地域**: 大字・丁目に対応する路線価図PDFを取得
- **倍率地域**: 市区町村単位の倍率表PDFを取得
- 宅地は路線価図を優先し、倍率地域の場合は倍率表に自動フォールバック
- 農地・山林など宅地以外は倍率表のみ取得
- 同一市区町村・大字の重複ダウンロードをキャッシュで防止

## 必要なもの

- Python 3.10以上

```bash
pip install -r requirements.txt
```

## CSVフォーマット

課税明細書PDFをClaude Chatに投げて以下のフォーマットでCSV変換してから使用してください。

```
年分,所在地,地目,地積,固定資産税評価額,課税標準額_固定資産税,課税標準額_都市計画税
r07,東京都千代田区丸の内1丁目1番地,宅地,150.00,50000000,,
r07,東京都西多摩郡奥多摩町〇〇番地,宅地,200.00,3600000,3600000,
r07,埼玉県秩父郡小鹿野町〇〇番地,田,400.00,800000,800000,
```

| 列 | 説明 |
|---|---|
| 年分 | 相続開始年分（`r07` = 令和7年）。省略時はデフォルト値を使用 |
| 所在地 | 都道府県から始まる完全な地番。郡名・政令市の区名を含んでいても可 |
| 地目 | 宅地・田・畑・山林・原野・雑種地など |
| 地積 | 面積（㎡） |
| 固定資産税評価額 | 数値のみ（カンマ不要） |
| 課税標準額_固定資産税 | 任意 |
| 課税標準額_都市計画税 | 任意 |

## 使い方

```bash
python pipeline.py <課税明細書CSV> [年分] [PDF保存先ディレクトリ]
```

```bash
# 基本（CSVの年分列を使用、PDFはoutputフォルダに保存）
python pipeline.py kazeimeisaisho.csv

# 年分を明示指定する場合（CSVに年分列がない場合）
python pipeline.py kazeimeisaisho.csv r07

# PDF保存先を変更する場合
python pipeline.py kazeimeisaisho.csv r07 ./my_output
```

実行すると以下の構造でPDFが保存されます。

```
output/
  rosenka/        ← 路線価図PDF（路線価地域の宅地）
    r07_東京都_千代田区_丸の内１_rosenka.pdf
    ...
  bairitu/        ← 倍率表PDF（倍率地域・農地・山林など）
    r07_東京都_奥多摩町_bairitu.pdf
    ...
  result.csv      ← 取得結果一覧
```

## scraper.py の単体利用

```bash
# 倍率表
python scraper.py bairitu 東京都 奥多摩町 '' r07 output

# 路線価図
python scraper.py rosenka 東京都 千代田区 丸の内1丁目 r07 output
```

## 注意事項

- 国税庁サイトへのアクセス間隔は1.0秒に設定しています（過負荷対策）
- 倍率表PDFは画像PDFのため、倍率の読み取りは目視で行ってください
- 路線価図は大字・丁目単位で取得するため、1筆に複数の路線価図が対応する場合は最初の1枚を取得します
