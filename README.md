# zaisan-hyoka

倍率地域の相続税評価において、課税明細書CSVを読み込み、国税庁サイトから市区町村ごとの倍率表PDFを一括ダウンロードするPythonスクリプトです。

## できること

- 課税明細書CSVを読み込み、住所から市区町村を自動判定
- 国税庁 財産評価基準書サイトから倍率表PDFを市区町村単位で一括ダウンロード

## 必要なもの

- Python 3.10以上
- requests
- beautifulsoup4

```bash
pip install requests beautifulsoup4
```

## CSVフォーマット

課税明細書PDFをClaude Chatに投げて以下のフォーマットでCSV変換してから使用してください。

```
年分,所在地,地目,地積,固定資産税評価額,課税標準額_固定資産税,課税標準額_都市計画税
r07,東京都西多摩郡奥多摩町〇〇番地,宅地,150.00,3600000,3600000,
r07,埼玉県秩父郡小鹿野町〇〇番地,田,400.00,800000,800000,
```

| 列 | 説明 |
|---|---|
| 年分 | 相続開始年分（`r07` = 令和7年）。省略時はデフォルト値を使用 |
| 所在地 | 都道府県から始まる完全な地番。郡名を含んでいても可 |
| 地目 | 宅地・田・畑・山林・原野・雑種地など |
| 地積 | 面積（㎡） |
| 固定資産税評価額 | 数値のみ（カンマ不要） |
| 課税標準額_固定資産税 | 任意 |
| 課税標準額_都市計画税 | 任意 |

## 使い方

```bash
python pipeline.py <課税明細書CSV> [年分] [PDF保存先ディレクトリ]
```

```bash
# 例：令和7年分、PDFをbairitu_pdfsフォルダに保存
python pipeline.py kazeimeisaisho.csv

# 年分を明示指定する場合（CSVに年分列がない場合）
python pipeline.py kazeimeisaisho.csv r07

# PDF保存先を変更する場合
python pipeline.py kazeimeisaisho.csv r07 ./output
```

実行すると `bairitu_pdfs/` フォルダ（デフォルト）に市区町村ごとの倍率表PDFが保存されます。

```
bairitu_pdfs/
  r07_東京都_奥多摩町.pdf
  r07_埼玉県_小鹿野町.pdf
  ...
```

倍率表PDFを確認し、以下の計算を行ってください。

```
相続税評価額 ＝ 固定資産税評価額 × 倍率
```

## 注意事項

- 国税庁サイトへのアクセス間隔は0.8秒に設定しています（過負荷対策）
- 倍率表PDFは画像PDFのため、倍率の読み取りは目視で行ってください
- 路線価地域（路線価が設定されている地域）には対応していません
